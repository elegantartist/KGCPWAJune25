# User Onboarding Flow

## 1. Sign Up

- The user signs up for an account with their email and password.
- The user is sent a verification email.
- The user clicks the link in the verification email to verify their account.

## 2. Welcome

- The user is shown a welcome message.
- The user is prompted to create their profile.

## 3. Profile Creation

- The user enters their name, age, and other personal information.
- The user sets their health goals.
- The user is shown a summary of their profile.

## 4. Feature Tour

- The user is given a tour of the application's features.
- The user is shown how to use the dashboard, log their daily scores, and chat with their doctor.

## 5. Dashboard

- The user is taken to their dashboard.
- The user can now start using the application.
