# Product Overview

KGC (Keep Going Care) is a comprehensive health coaching Progressive Web Application (PWA) that connects patients, doctors, and administrators in a unified care management platform.

## Core Purpose
- **Patient Care Management**: Patients track daily health scores (diet, exercise, medication adherence)
- **AI-Powered Support**: Enhanced chatbot provides 24/7 guidance and motivation
- **Doctor Oversight**: Healthcare providers monitor patient progress and set Care Plan Directives (CPDs)
- **Gamified Motivation**: Progress milestones system with badges and financial rewards

## Key User Roles
- **Patients**: Primary users who track health metrics, interact with AI chatbot, and access wellness resources
- **Doctors**: Healthcare providers who monitor patients and configure care plans
- **Admins**: System administrators with full platform oversight

## Core Features
- Daily self-scoring system for health adherence
- AI chatbot with multiple provider support (OpenAI, Anthropic, Grok)
- Motivational image processing and "Keep Going Sequence"
- Inspiration machines for diet and exercise content
- Progress tracking with gamification elements
- Real-world support integration (gyms, meal delivery, pharmacy pricing)

## Technical Approach
- Offline-first PWA design with connectivity awareness
- Privacy-focused with PII/PHI protection
- Multi-AI provider architecture for resilience
- Responsive design optimized for mobile and desktop
